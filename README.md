# Nayepankh Foundation Website

## Overview

This is the official website of Nayepankh Foundation, developed using HTML, CSS, and JavaScript. The website provides information about the foundation, its programs, gallery, volunteering opportunities, donations, and contact details.

## Features

* Responsive Homepage
* About Us Section
* Programs Page
* Gallery Page
* Volunteer Registration
* Donation Information
* Contact Page
* Common Footer on All Pages

## Installation

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Open the project folder.

3. Run `index.html` using Live Server or any local server.
